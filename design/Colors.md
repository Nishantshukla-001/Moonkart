# MoonKart Color System

## Purpose

This document defines the official color palette and color usage guidelines for MoonKart.

MoonKart should have a soft, elegant, premium, feminine, and luxury appearance inspired by boutique fashion and lifestyle brands.

Every page, component, dashboard, and user interface element must follow this color system to maintain consistency throughout the application.

Random colors should never be introduced without updating this document.

---

# Design Theme

MoonKart should feel:

- Premium
- Feminine
- Elegant
- Modern
- Soft
- Minimal
- Luxury
- Boutique Style
- Clean
- Trustworthy

The design should focus on simplicity, generous whitespace, rounded components, and pastel colors rather than bright commercial colors.

# Gradients

Use very subtle gradients only.

Example

Soft Sage Green → White

Soft Blush Pink → White

Warm Yellow → White

Gradients should never be loud or highly saturated.

---

# Color Usage Rules

- White should remain the dominant color.
- Sage Green is the primary action color.
- Warm Yellow is reserved for featured and promotional sections.
- Blush Pink should only be used as an accent.
- Never use bright neon colors.
- Maintain generous whitespace.
- Keep the design calm and premium.
- Use colors consistently across all pages.

---

# Accessibility

- Maintain WCAG AA color contrast.
- Never rely on color alone to convey meaning.
- Pair important status colors with icons or labels.
- Ensure buttons have clear hover and focus states.

---

# Design Philosophy

MoonKart should feel like a premium fashion and lifestyle boutique.

The interface should communicate elegance, warmth, softness, and trust through a carefully balanced pastel palette.

Every page should feel clean, spacious, modern, and luxurious while remaining easy to use and visually consistent.
