# MoonKart Typography System

## Purpose

This document defines the typography system for MoonKart.

Typography should create a premium, elegant, modern, and feminine shopping experience.

The website should feel clean, luxurious, readable, and visually balanced.

Typography should remain consistent across every page.

---

# Typography Philosophy

MoonKart typography should communicate:

- Premium Quality
- Soft Luxury
- Modern Fashion
- Elegance
- Simplicity
- Readability

Avoid overly decorative fonts.

Focus on clean typography with excellent spacing.

---

Typography Rules

- Never use more than two font families.
- Maintain consistent spacing.
- Keep headings bold and clean.
- Use generous whitespace.
- Avoid decorative fonts.
- Maintain clear visual hierarchy.
- Ensure readability on all screen sizes.
- Keep line lengths comfortable for reading.

---

Accessibility

- Use sufficient font sizes.
- Maintain proper line spacing.
- Ensure adequate contrast.
- Never use extremely thin fonts for important information.
- Follow WCAG readability guidelines.

---

Final Typography Philosophy

MoonKart typography should make users feel they are browsing a premium boutique.

The text should be elegant, calm, modern, highly readable, and visually balanced.

Every heading, paragraph, label, and button should reinforce the premium identity of the platform.
